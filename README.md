# 🎨 Mondrian Painting Project

A recreation of the iconic **Piet Mondrian** abstract painting using only **HTML** and **CSS Grid**. This project demonstrates the power of CSS Grid by accurately reproducing a famous geometric artwork without using images.

## 📖 About

This project recreates the classic **Composition with Red, Blue and Yellow** style by Piet Mondrian. Every colored block and gap is built using CSS Grid, showcasing how modern CSS can be used to create complex visual layouts with precision.

It's a great exercise for understanding grid tracks, spanning, positioning, and spacing.

## 🚀 Features

* 🎨 CSS Grid-based artwork
* 📐 Accurate geometric layout
* 🟥 Classic Mondrian color palette
* 🖼️ No images used — built entirely with HTML & CSS
* ✨ Clean and minimal code structure

## 🛠️ Technologies Used

* HTML5
* CSS3 (Grid)

## 🎯 What I Practiced

* Creating layouts with CSS Grid
* Using `grid-template-rows` and `grid-template-columns`
* Positioning items with `grid-row`, `grid-column`, and `grid-area`
* Working with gaps and proportions
* Recreating real-world designs using only CSS
